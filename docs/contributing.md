# Contributing

```
https://github.com/danielpacak/awesome-threat-detection.git
cd awesome-threat-detection
```

Use standalone installer or choose any [other](https://docs.astral.sh/uv/getting-started/installation/) installation method that works for you.

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

(Optional) Update uv to the latest version if it's already installed.

```
uv self update
```

```
uv sync --frozen
```

```
uv run mkdocs serve
```

Open <http://localhost:8000/awesome-threat-detection/> in your web browser.
